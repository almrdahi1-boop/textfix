# textfix.net - مدقق إملائي ونحوي مجاني بالذكاء الاصطناعي ✏️

**تصحيح النصوص العربية والإنجليزية فوراً** — أداة مجانية للتدقيق الإملائي والنحوي باستخدام Llama 3.3 70B.

🔗 https://textfix.net

## المميزات
- 🤖 ذكاء اصطناعي (Llama 3.3 70B) لتصحيح دقيق
- 🌍 يدعم العربية والإنجليزية
- 🔄 تحويل الأحرف: Capitalize, UPPERCASE, lowercase, Title Case
- 💯 مجاني بالكامل — بدون حدود أو تسجيل
- ⚡ تصحيح فوري خلال ثوانٍ

## التقنيات
- **Frontend**: HTML + CSS + Vanilla JS
- **AI API**: Groq (Llama 3.3 70B)
- **Hosting**: Vercel + Cloudflare
